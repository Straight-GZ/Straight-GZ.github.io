  2. CSS normalize 文档

    • @import-normalize;
    • 作用是让不同浏览器上的默认样式相近

  3. 支持 sass文档

      1. node-sass 下载速度慢，本地编译慢，使用 dart-sass 代替，但 React 不支持 dart-sass
    • yarn add node-sass@npm:dart-sass:使用 dart-sass 代替 node-sass github
    • 或者在 package.json 配置，修改 dependencies 配置
    • npm6.9 新功能 package alias
